A.4 Configuration Validation Logic

   Algorithm: Computer Configuration Validator

     1: Input: User configuration C, Purpose P
     2: Output: Valid configuration or error message
     3: Apply purpose rules:
     4: if P == "training" then
     5: C.GPU  PR.H200
     6: else if P == "inference" then
     7: C.GPU  PR.A10G
     8: end if
     9: Check physical compatibility:
    10: if C.CPU.socket = C.Motherboard.socket then
    11: return Error: "CPU-Motherboard socket mismatch"
    12: end if
    13: Check power requirements:
    14: total power  C.CPU.TDP + C.GPU.TDP + 100 {100W buffer}
    15: if total power > C.PSU.wattage then
    16: return Error: "Insufficient PSU capacity"
    17: end if
    18: Check interface compatibility:
    19: if C.GPU.interface = C.Motherboard.PCIe version then
    20: return Error: "GPU interface mismatch"
    21: end if
    22: Check RAM compatibility:
    23: if C.RAM.type = C.Motherboard.RAM type then
    24: return Error: "RAM type not supported"
    25: end if
    26: return "Configuration valid"
