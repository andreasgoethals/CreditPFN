# =============================================================================
# CreditPFN — runtime dependencies

# Used by every script in src/data/ and (later) src/train, src/eval, src/model.
# Pinned to conservative major-version ranges. We version-pin minor where the
# upstream API has changed in a breaking way recently.
#
# Python version: ** USE 3.11 OR 3.12 ** (do NOT use 3.14)
# -----------------------------------------------------------------------------
# scikit-learn, torch, and several other compiled packages do not yet ship
# prebuilt wheels for Python 3.14 (cp314). Pip falls back to compiling from
# source via MSVC, which crashes mid-build on Windows. Python 3.12 is the
# sweet spot: every dep below has a cp312 wheel on PyPI as of late 2025.
#
# To switch: install Python 3.12 from python.org, then
#     py -3.12 -m venv .venv --prompt CreditPFN
#     .venv/Scripts/activate
#     python -m pip install --upgrade pip
#     pip install -r requirements.txt
#
# Refresh with:  pip install -r requirements.txt
# =============================================================================
