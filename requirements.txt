# Pinned to conservative major-version ranges. Audited 2026-05-27 against the
# actual `import` statements in src/ and the verified TabPFN API in
# `repositories/TabPFN .txt`.
#
# Python version: ** USE 3.11 OR 3.12 ** (do NOT use 3.14)
# -----------------------------------------------------------------------------
# scikit-learn, torch, and several other compiled packages do not yet ship
# prebuilt wheels for Python 3.14 (cp314). Pip falls back to compiling from
# source via MSVC, which crashes mid-build on Windows. Python 3.12 is the
# sweet spot: every dep below has a cp312 wheel on PyPI as of late 2025.
#
# Quick start:
#     py -3.12 -m venv .venv --prompt CreditPFN          # Windows
#     python3.12 -m venv .venv --prompt CreditPFN        # Linux / macOS
#     source .venv/bin/activate                          # Linux / macOS
#     # .venv/Scripts/activate                           # Windows
#     python -m pip install --upgrade pip
#     pip install -r requirements.txt
#
#     # IMPORTANT: TabPFN's PyPI release lags behind main. Override with
#     # the git source so we get the v2.6/v3 API our code relies on:
#     pip install --upgrade "tabpfn @ git+https://github.com/PriorLabs/tabPFN.git@main"
# =============================================================================

# --- Core deep-learning stack ------------------------------------------------
torch>=2.2,<3.0

# --- TabPFN ------------------------------------------------------------------
# This project's code is written against the TabPFN API documented in
# `repositories/TabPFN .txt` (verified 2026-05-27). Key surface used:
#   * load_model_criterion_config returns 4-tuple
#     (list[Architecture], criterion, list[ArchitectureConfig], InferenceConfig)
#   * version: Literal["v2", "v2.5", "v2.6", "v3"]
#   * download_if_not_exists keyword
#   * PerFeatureTransformer.forward(x, y, *, only_return_standard_out,
#     categorical_inds=list[list[int]])  — auto-derives single_eval_pos from len(y)
#   * tabpfn.preprocessing.ensemble.TabPFNEnsemblePreprocessor (NOT re-exported
#     from the top-level tabpfn.preprocessing — verified at TabPFN .txt:5876)
#   * FeatureSchema.from_only_categorical_indices(cats, num_columns=N)
#     (TabPFN .txt:30229-30233)
#   * get_resolved_outlier_removal_std(estimator_type="classifier" | "regressor")
#     (TabPFN .txt:10622-10637)
#
# PyPI's `tabpfn` historically lagged main by a few months. Bump the floor
# whenever PyPI catches up to a known-good release. The git-source install
# noted above bypasses PyPI entirely.
